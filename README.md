# Sales-Dashboard-Python📊 

An interactive Sales Dashboard built using Python, Dash, and Plotly to visualize sales trends, regional performance, and category-wise insights with dynamic filtering functionality.

---

# 🚀 Features

✅ Interactive Dashboard UI  
✅ Dynamic Region Filter  
✅ Sales Trend Analysis  
✅ Region-wise Sales Comparison  
✅ Category-wise Sales Distribution  
✅ Real-Time Graph Updates  
✅ Data Visualization using Plotly  
✅ Dashboard Development using Dash

# 📦 Installation

## 1️⃣ Clone Repository

```bash

```

---

## 2️⃣ Move into Project Folder

```bash
cd interactive-sales-dashboard
```

---

## 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

## If Using Jupyter Notebook

Run:

```python
app.run_server(mode="inline")
```

---

## If Using VS Code / Terminal

Replace:

```python
app.run_server(mode="inline")
```

with:

```python
app.run_server(debug=True)
```

Then run:

```bash
python app.py
```

---

# ⚙️ How It Works

1. Random sales dataset is generated using NumPy and Pandas.
2. Dash creates the web dashboard layout.
3. Plotly generates interactive charts.
4. Dropdown filter updates charts dynamically using Dash callbacks.
5. Dashboard runs locally in Jupyter Notebook or browser.
